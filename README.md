# 📩 Spam Detection using NLP and Machine Learning

## 📌 Overview

This project is an SMS Spam Detection system built using **Python**, **Natural Language Processing (NLP)**, and **Machine Learning**. It classifies SMS messages as **Spam** or **Ham (Not Spam)** using text preprocessing techniques and a **Multinomial Naive Bayes** classifier.

## 🚀 Features

* SMS Spam Classification
* Text Preprocessing using NLP
* TF-IDF Vectorization
* Multinomial Naive Bayes Model
* Model Evaluation
* Real-time Message Prediction
* Data Visualization with Graphs

## 🛠️ Technologies Used

* Python
* Pandas
* Matplotlib
* NLTK
* Scikit-learn

## 🧠 NLP Techniques

* Lowercasing
* Punctuation Removal
* Stopword Removal
* Stemming
* TF-IDF Vectorization

## 📊 Visualizations

## 🤖 Machine Learning Model

